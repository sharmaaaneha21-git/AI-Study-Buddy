# 🤖 AI Chat Assistant (Python Project)

A modular AI Chat Assistant built using Python.  
This project includes Terminal, GUI, and Web-based chatbot implementations.

The goal of this project is to explore backend logic, GUI development, and web application development using Python.

This project is structured in a scalable way so that OpenAI API can be integrated in the future for advanced conversational AI capabilities.

---

## 🚀 Features

- Rule-based chatbot logic
- Text-to-Speech support (pyttsx3)
- Terminal-based chatbot
- GUI chatbot using Tkinter
- Web-based chatbot using Flask
- Chat history saving
- Clean modular folder structure
- Future-ready for OpenAI API integration

---

## 🧠 Tech Stack

- Python
- Flask
- Tkinter
- pyttsx3
- HTML (for Flask templates)

## 💻 Versions Included

### 2️⃣ GUI Version

- Ready for deployment structure

## ⚙️ Installation

Install required libraries:

pip install flask  
pip install pyttsx3  

---

## ▶️ How to Run

### Terminal Version
python main.py

### GUI Version
python gui_chat.py

### Flask Version
python app.py

Then open in browser:
http://127.0.0.1:5000
